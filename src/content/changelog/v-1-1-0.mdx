This version focuses on making the developer and user experience more robust and resilient.

### Improvements

- Added global and route-specific loading skeletons (`loading.tsx`).
- Created custom `error.tsx` and `not-found.tsx` pages for graceful error handling.

### Bug Fixes

- Resolved Contentlayer schema warnings for the `Doc` type by adding a `date` field.
- Fixed runtime errors related to incorrect `params` usage in dynamic routes.
