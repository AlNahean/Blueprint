Welcome to the first version of this starter template! This release includes all the core features needed to build a modern, content-driven website.

### Core Features

- **Framework:** Next.js 15 (App Router) & React 19
- **Styling:** Tailwind CSS v4 & shadcn/ui
- **Content:** MDX powered by Contentlayer for blogs, docs, and pages
- **SEO:** Dynamic metadata, sitemaps, and `robots.ts`
